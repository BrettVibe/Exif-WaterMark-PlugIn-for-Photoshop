## 快速开始

1. 确认 Photoshop 版本 ≥ 2025（v26.0+）
2. 双击 `exif-watermark-plugin-v1.1.0.ccx` 安装
3. 重启 Photoshop
4. 菜单「窗口 › 扩展（插件）」→「EXIF 水印」
5. 打开照片，勾选字段，点击「一键插入 EXIF 水印」

详细步骤见 [安装与使用指南.md](./docs/user/安装与使用指南.md)。
